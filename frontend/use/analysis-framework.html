🎯 研究问题定义
核心问题：中国心血管疾病疾病负担现状如何？主要风险因素是什么？未来趋势如何？

研究假设：
1. 中国心血管疾病DALYs率呈上升趋势，但增速可能放缓
2. 高血压、吸烟、不健康饮食是主要可干预风险因素
3. 城乡、区域间存在显著健康不公平性
4. 人口老龄化将加剧心血管疾病负担
📊 分析方法体系
描述性统计分析
计算发病率、患病率、死亡率等指标，描述疾病分布特征
趋势分析（Joinpoint）
识别疾病负担变化趋势转折点，量化年度变化百分比(APC)
归因风险分析
计算人群归因分数(PAF)，识别主要可干预风险因素
预测模型（BSTS）
贝叶斯结构时间序列模型，预测未来疾病负担趋势
健康公平性分析
计算集中指数、斜率指数，评估健康不公平程度
空间聚类分析
识别疾病高发聚集区域，为精准干预提供依据

📋 核心分析结论
  • 疾病负担严峻：中国心血管疾病患病人数达3.3亿，占总人口的23.3%，是全球最大的心血管疾病负担国
  • 死亡率下降趋势：年龄标准化死亡率从2010年的499/10万降至2024年的458/10万，降幅8.3%，但绝对死亡人数仍居高位
  • 风险因素明确：高血压、吸烟、高盐饮食是最主要的可干预风险因素，三者合计归因分数达97.5%
  • 城乡差异显著：农村地区心血管疾病死亡率是城市的1.3倍，医疗资源配置不均问题突出
  • 老龄化挑战：65岁以上人群心血管疾病患病率是总人口的3.2倍，人口老龄化将加剧疾病负担
  • 防控成效初显：成人吸烟率下降、高血压知晓率提升，综合防控措施开始显现效果
💡 基于证据的决策建议
🔴 最高优先级
高血压防控：推进"知晓您的血压"行动，提高高血压知晓率(当前51.6%)、治疗率(当前45.8%)和控制率(当前16.8%)。建议将血压筛查纳入基本公共卫生服务。
🔴 最高优先级
控烟行动：全面落实《"健康中国2030"规划纲要》控烟目标，提高烟草税，推进无烟环境立法，加强戒烟服务体系建设。
🟡 高优先级
减盐行动：推广低钠盐，推动食品工业减盐，开展全民减盐健康教育，目标人均每日食盐摄入量从9.3克降至5克以下。
🟡 高优先级
体重管理：实施"体重管理年"活动，推广健康生活方式，加强儿童青少年肥胖防控，遏制肥胖率上升趋势。
🟢 中优先级
基层能力提升：加强基层医疗机构心血管疾病防治能力，推进分级诊疗，实现"小病不出村、大病不出县"。
🟢 中优先级
健康公平：缩小城乡、区域、人群间健康差距，重点加强中西部农村地区心血管疾病防控资源投入。
