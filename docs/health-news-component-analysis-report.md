## 1. 组件功能概述

### 1.1 功能定位

健康资讯组件是"健康数据洞察平台"首页的重要功能模块，位于页面最底部，负责展示最新的健康、医疗相关新闻资讯。该组件通过与第三方 Mediastack API 集成，实现了动态内容更新，为用户提供时效性强的健康信息。

### 1.2 核心功能

| 功能模块 | 说明 | 优先级 |
|---------|------|--------|
| **动态内容获取** | 从 Mediastack API 获取实时健康新闻 | P0 |
| **24小时缓存** | 智能缓存机制，避免频繁API调用 | P0 |
| **API调用监控** | 严格监控每月100次API调用限额 | P1 |
| **错误兜底** | 网络异常时显示静态备用内容 | P1 |
| **搜索过滤** | 支持关键词搜索过滤资讯 | P2 |
| **点击跳转** | 点击资讯卡片跳转到原文链接 | P2 |

```json

  "source": "新闻来源",
```

### 3.3 缓存策略详解

| 策略 | 说明 | 实现方式 |
|------|------|----------|
| **时间策略** | 24小时自动过期 | Redis TTL / 内存时间戳 |
| **空间策略** | 按天存储 | 缓存键: `mediastack:news:2026-04-18` |
| **降级策略** | Redis不可用时用内存 | 自动检测Redis连接状态 |
